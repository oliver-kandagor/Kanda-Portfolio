# Oliver Kandagor Koech - Portfolio

A clean, modern, and fully responsive portfolio website showcasing my work as a Full Stack Developer and founder of Shujaa Code.

## 🚀 Features

- Pure HTML and CSS implementation (no JavaScript)
- Fully responsive design for all devices
- Smooth scrolling and CSS animations
- Clean and modern UI
- Contact form
- Downloadable CV
- Social media integration

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome Icons

## 🌟 Sections

- Home/Hero
- About Me
- Educational Background
- Skills
- Projects
- Interests
- Contact Form

## 🔧 Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/portfolio.git
   ```

2. Open `index.html` in your web browser
